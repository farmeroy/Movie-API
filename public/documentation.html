Movie API Documentation

This is a REST API that:

  • Will provide users with access to information about different movies (the directors, genres, year produced etc.)
  • Can provie a list of all movies in the Database
  • Users will be able to:
    • Create an account
    • Update or edit their personal information
    • Create a list of their favorite movies
    • Delete their account
Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None JSON object contain all movie data
Return data about a single movie (by title) /movies/[title] GET None JSON object containing the data abou the requested movie.
Return the genre of a movie (by title) /movies/[title]/genre GET None A text message with the movie name and genre.
Return information about the director of a movie /movies/:Title/director GET None JSON object containing data about the director.
Return data about a single director (by name) /directors/[name] GET None JSON obect containing data about the director.
Return information about all users /users GET None An array of user objects (JSON)
Return information about a single user by name /users/[Username] GET none JSON object with user info
Register new User /users POST JSON object should look like this:
{ "Username": [String], "Password": [String], "Email": [String], "Birthday": [Date] }
The user as a JSON object is returned
Update user info /users/[userName]/update PUT A JSON object which sends new values for any (or all) of these keys
{ "Username": [String], "Password": [String], "Email": [String], "Birthday": [Date] }
A JSON object containing all current user info is returned.
Delete User Account /users/[username]/delete DELETE None Text confirmation that the user account was deleted
Add film to User's favorite movie list /users/[username]/movies/[MovieID] PUT None The movie id is returned as confirmation.
Remove a film from the User's film list /users/[Username]/movies/remove/[MovieID] PUT None Text confirmation that the movie was deleted
